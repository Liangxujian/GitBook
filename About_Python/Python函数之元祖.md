# Python 函数之元祖

* 元组可以使用下标索引来访问元组中的值
* **元组中的元素值是不允许修改的**，但我们可以对元组进行连接组合
* **元组中的元素值是不允许删除的**，但我们可以使用 del 语句来删除整个元组
* 与字符串一样，元组之间可以使用 + 号和 * 号进行运算，即可以组合和复制

| 函数       | 描述                 |
| ---------- | -------------------- |
| len(tuple) | 计算元组元素个数     |
| max(tuple) | 返回元组中元素最大值 |
| min(tuple) | 返回元组中元素最小值 |
| tuple(seq) | 将列表转换为元组     |
